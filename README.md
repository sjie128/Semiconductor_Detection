🔬 Semiconductor Reliability Guard: AI-Powered Thermal Analyzer
Elevate Your Failure Analysis Workflow
Transform standard imaging into a high-precision Reliability Dashboard. This system doesn't just "look" at hardware—it interrogates it using a dual-engine AI approach to identify thermal hotspots and structural integrity in real-time.

🚀 Core Capabilities
⚡ Real-Time YOLO Integration: Harnesses the speed of YOLOv8 to pinpoint semiconductors instantly with high confidence.

🔥 Thermal Stress Quantification: Automatically calculates Stress Index (MPa) based on heat spread ratios, predicting potential failure points before they occur.

🛠️ AI Structural Auditing: Goes beyond temperature to inspect physical form, flagging cracks and defects with an edge-density analyzer.

🎨 Vibrant Pseudocolor Mapping: Translates raw data into high-contrast Plasma and Jet colormaps for intuitive, industrial-grade visualization.

📊 Integrated Terminal Logging: Generates a continuous reliability report for audit trails and manufacturing quality control.

🎮 Interactive Control Logic
c: Cease Scanning – Instantly halts the AI engine and saves the current log state.

q: Quit Dashboard – Safely releases the hardware camera and cleans up the memory environment.

🛠️ Technical Stack
Engine: Python 3.11

AI Models: Ultralytics YOLOv8 (Nano for low-latency performance)

Computer Vision: OpenCV (HSV color space mapping & Pseudocoloring)

Mathematics: NumPy (Pixel-density distribution & Stress calculations)
